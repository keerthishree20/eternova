# Guide — Eternova

**Live:** https://eternova-peach.vercel.app

## Prerequisites

- Python 3.10+ (for Python projects) / Node.js 18+ (for Next.js projects)
- Git

## Setup

```bash
# Clone the repository
git clone https://github.com/keerthishree20/eternova.git
cd eternova
```

## Running the App

### Backend (FastAPI)
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
API runs at `http://localhost:8000`

### Frontend (Next.js)
```bash
cd frontend
npm install
npm run dev
```
App runs at `http://localhost:3000`
